Theme and Theme Editor

For XMind, theme is the collection of colors, shapes, line patterns, and other coordinated graphical attributes of a mind map. We can apply the theme with one step to change the whole mind map's style. Also, theme can be further customised and saved for future use with other maps. By default, XMind has multiple well-designed theme.

To Change Theme
  1. Open Theme View by clicking "View-Theme".
  2. Double-click the theme you like.
To Extract a Theme
  1. Design your own theme by choosing color, shape, line, background.
  2. Choose " Tools > Extract Theme..." on the menu.
  3. Name this new theme in the "Theme View".
  4. Now you can apply this theme to other maps.

Note: Every theme can only use one kind of topic style, boundary style.

Set the default Theme

Every blank mind map is applied the default theme. And we can set any theme as the "Default theme" freely.

  1. Open the theme view, and click the target theme.
  2. Click the star icon at the upper right corner of the theme view.

Theme editor

We can further design the theme with theme editor, and preview it seamlessly.

  1. Open the theme view by clicking "View - Theme"
  2. Right clicking on the target theme, and choose "Edit the theme"
  3. Select the objective at the upper right corner of the dialog, and change the corresponding properties settings
  4. Click “Save as a new theme”
